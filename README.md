# 哈尔滨太™热了 / Harbin is so f**king hot

## 🌡️ 项目简介

**给高考生的真实参考 | 给在校生的吐槽平台**

哈尔滨，一个被大家认为"很凉快"的城市，但现实是夏天真的很热！本项目收集哈尔滨各大高校的**真实生活环境信息**，帮助高考填报志愿的同学了解：

- 🏠 **宿舍真实环境**：温度、通风、空调情况
- ❄️ **降温设施**：空调安装情况、使用政策、费用
- 📝 **反馈处理**：学校对学生环境投诉的处理情况
- 💰 **生活成本**：夏季额外的降温费用
- 🎓 **学习环境**：教室、图书馆、实验室的温控情况

## 📊 信息分类

### 🏠 宿舍环境
- **温度实测**：夏季宿舍内实际温度记录
- **空调情况**：有无空调、型号、制冷效果
- **通风条件**：窗户朝向、风扇数量、空气流通
- **入住体验**：不同楼层、不同朝向的差异

### ❄️ 降温设施
- **空调政策**：学校空调安装计划、使用规定
- **费用情况**：电费、空调租赁费、维修费
- **替代方案**：学校提供的其他降温措施
- **DIY降温**：学生自己的降温神器分享

### 📋 投诉反馈
- **投诉渠道**：学校的官方反馈途径
- **处理效率**：从投诉到解决的时间
- **处理结果**：学校的实际改善措施
- **态度评价**：相关部门的服务态度

### � 学习环境
- **教室温控**：各教学楼的空调情况
- **图书馆环境**：自习区域的温度舒适度
- **实验室条件**：特殊环境的温控要求
- **考试环境**：期末考试期间的教室温度

### � 生活成本
- **电费支出**：夏季用电成本增加
- **降温用品**：风扇、凉席等采购建议
- **饮食变化**：夏季饮食成本变化
- **其他开销**：因高温产生的额外费用

## 🛠️ 开发和部署

### 自动构建
本项目使用 GitHub Actions 实现自动构建和部署：

- ✅ **自动触发**：每次推送到 main/master 分支
- ✅ **数据分析**：自动运行环境数据分析工具
- ✅ **网站构建**：生成静态网站到 `dist/` 目录
- ✅ **自动部署**：部署到 GitHub Pages

### 本地构建

#### Linux/macOS
```bash
# 运行构建脚本
chmod +x build.sh
./build.sh

# 或手动构建
python tools/environment_analyzer.py .
mkdir -p dist
cp index.html dist/
cp -r data dist/
```

#### Windows
```cmd
REM 运行构建脚本
build.bat

REM 或手动构建
python tools\environment_analyzer.py .
mkdir dist
copy index.html dist\
xcopy data dist\data\ /e /i
```

### 本地预览
```bash
cd dist
python -m http.server 8000
# 访问 http://localhost:8000
```

### 数据流程
1. **收集** → 学生通过 Issues/邮箱提交信息
2. **整理** → 维护者将数据分类到对应目录
3. **分析** → 自动运行数据分析生成报告
4. **构建** → GitHub Actions 自动构建网站
5. **部署** → 自动部署到 GitHub Pages
