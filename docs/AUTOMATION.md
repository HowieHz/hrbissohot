# 🤖 项目自动化功能说明

## 🎯 自动化目标

"哈尔滨太™热了"项目通过GitHub Actions实现了完全自动化的数据收集、处理和展示流程。

## 🔄 自动化流程

### 1. 数据收集自动化
```
用户提交Issue → 自动标签分类 → 人工审核 → 数据整理 → 报告更新
```

### 2. 网站部署自动化
```
代码推送 → 数据分析 → 网站构建 → GitHub Pages部署
```

### 3. 数据更新自动化  
```
定时任务 → 扫描数据文件 → 重新分析 → 生成报告 → 自动提交
```

## 📊 自动生成的内容

### 数据分析报告
- **汇总报告** (`data/summary_report.md`)
- **JSON数据** (`data/environment_data.json`)  
- **每日报告** (`data/daily_report_YYYY-MM-DD.md`)
- **实时统计** (`data/live_stats.md`)
- **项目仪表板** (`data/dashboard.md`)

### 网站文件
- **主页面**: 自动更新统计数据
- **API接口**: JSON格式数据接口
- **站点地图**: 自动生成的页面索引

## 🚀 部署流程

### GitHub Pages自动部署
1. **触发**: 推送到主分支
2. **构建**: 运行数据分析，生成网站文件
3. **部署**: 自动发布到GitHub Pages
4. **访问**: 通过`username.github.io/hrbissohot`访问
