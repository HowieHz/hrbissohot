# 贡献指南 / Contributing Guide

欢迎为"哈尔滨太™热了"项目贡献真实环境信息！

## 🎯 项目目标

- 为高考生提供真实的校园环境参考
- 推动学校改善学生生活条件  
- 建立学生环境反馈的有效渠道
- 分享实用的生活经验和降温建议

## 📝 如何贡献

### 1. 环境信息提交

#### 我们需要的信息类型
- **🏠 宿舍环境**：温度实测、通风情况、生活体验
- **❄️ 降温设施**：空调安装情况、使用政策、费用
- **📋 投诉反馈**：学校处理态度、改善措施、时间进度
- **💰 生活成本**：因高温产生的额外开销

#### 信息要求
- ✅ 真实准确，有具体数据支撑
- ✅ 标注明确的时间和地点
- ✅ 保护个人隐私，可匿名提交
- ✅ 语言客观，避免情绪化表达

### 2. 提交格式模板

```markdown
# [学校简称] - [具体问题]

## 基本信息
- **学校**：[完整学校名称]
- **校区**：[如有多个校区请注明]
- **时间**：[信息对应时间，如2024年7月]
- **分类**：[宿舍环境/降温设施/投诉反馈/生活成本]

## 详细信息
[具体描述，数据越详细越好]

## 温度数据（如适用）
| 日期 | 室外温度 | 室内温度 | 测量时间 | 天气 |
|------|----------|----------|----------|------|
| XX月XX日 | XX°C | XX°C | XX点 | 晴/阴/雨 |

## 费用信息（如适用）
- 具体费用：XX元
- 费用说明：[用途和计算方式]

## 处理情况（如适用）
- 投诉时间：XXXX年XX月
- 处理结果：[学校具体回应]
- 满意度：⭐⭐⭐⭐⭐ (1-5星)

## 验证信息
- **信息来源**：[亲身经历/室友反馈/官方通知等]
- **可信度**：⭐⭐⭐⭐⭐ (1-5星)
- **提交者**：[年级专业或匿名]
- **更新时间**：XXXX年XX月
```

### 3. 提交方式

#### 方式一：GitHub Issues ⭐推荐⭐
1. 访问项目页面 [Issues](../../issues)
2. 点击 "New Issue"
3. 选择 "环境信息提交" 模板
4. 按格式填写后提交

#### 方式二：Pull Request
1. Fork 本项目到你的GitHub
2. 在对应学校文件夹下创建 `.md` 文件
3. 文件名格式：`YYYY_topic_description.md`
4. 提交 Pull Request

#### 方式三：邮件提交
- 邮箱：[待设置]
- 主题：[学校简称] 环境信息提交

### 5. 图片和证据

如有相关图片或文档：
- 📷 温度计读数照片
- 📋 学校回复截图  
- 📊 电费单据（隐去个人信息）
- 🏠 宿舍环境照片

请通过以下方式提供：
- 上传到图床后提供链接
- 邮件附件发送
- 在Issue中贴图

### 6. 内容审核标准

#### ✅ 欢迎的内容
- 客观真实的环境描述
- 具体的温度和费用数据
- 学校官方回应记录
- 实用的降温经验分享
- 建设性的改善建议

#### ❌ 不被接受的内容
- 恶意抹黑或夸大事实
- 涉及具体个人隐私
- 政治敏感话题
- 情绪化的抱怨内容
- 虚假或过时信息

### 7. 数据使用说明

- 📊 所有数据用于统计分析和报告生成
- 🔄 信息将在项目网站公开展示
- 📝 可能被媒体或研究机构引用
- 🔒 个人隐私信息会被妥善保护

### 8. 奖励机制

为感谢贡献者：
- 🏆 优质内容会在项目中特别标注
- 📈 积极贡献者可成为项目维护者
- 🎁 定期评选"最有价值贡献"

### 9. 联系我们

有任何问题或建议：
- 💬 GitHub Issues（公开讨论）
- 📧 邮箱（私人咨询）
- 🐛 Bug反馈和功能建议

感谢所有愿意分享真实经历的同学！你们的贡献将：
- 🎓 帮助学弟学妹做出明智选择
- 🏫 推动学校改善环境条件
- 📊 形成有价值的数据资料
- 🤝 建立学生互助网络
